Documentação Oficial

Instalação

Para instalar o Django, você pode utilizar o pip, o gerenciador de pacotes do Python:

pip install django

Você também pode criar um ambiente virtual para isolar as dependências do seu projeto:

python -m venv env
source env/bin/activate

Estrutura do Projeto

Um projeto Django normalmente é dividido em:

  • manage.py: arquivo de gerenciamento do projeto.
  • settings.py: configurações globais.
  • urls.py: roteamento de URLs.
  • apps/: onde ficam os módulos da aplicação.

Componentes Principais

O Django segue o padrão MTV:

  • Model: Representa os dados e regras de negócio.
  • Template: HTML com placeholders dinâmicos.
  • View: Lógica que responde às requisições.

Essa separação facilita a manutenção, reuso e escalabilidade do projeto.
